    3D // OpenGL // C++

    3D OpenGL is a pretty hard to understand engine, in which you can add, change, move objects around the scene, objects that can have different colors, textures, dimensions, or objects that can have their size changed from the scene. Lights and materials are important features for a game scene, features that can give your game a better perspective of a scene. Creating a game, using OpenGL, is hard because you cannot chose a perfect size of an object, or a perfect texture for the object
